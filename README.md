# yml-utils
🤞前端常用函数工具库

日常开发过程中会经常用到的`浏览器类型判断`、`日期格式化`、`数据处理`等常用函数，为避免不同项目使用，这里统一封装。并且使用 jest 进行测试，确保函数工具的正确性

### ✨使用

#### npm

```powershell
npm install yml-utils -S
```
